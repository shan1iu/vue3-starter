## vue3-starter

Node.js 16.13.0
pnpm 6.20.4

> based on [create-vue](https://github.com/vuejs/create-vue)

- vue3
- typescript
- vue-router
- pinia
- naive-ui
- tailwindCSS

- husky
- commitlint
- lint-staged

> start

```bash
# git
git init
# install deps
pnpm i
# run project
pnpm dev
```

> recommended

.vscode/settings.json

```json
{
	"editor.formatOnSave": true,
	"editor.codeActionsOnSave": {
		"source.fixAll.eslint": true
	},
	"vetur.validation.script": false
}
```
